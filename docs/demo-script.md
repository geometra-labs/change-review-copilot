# Demo Script

## Goal

Show that Change Review Copilot helps an engineer understand what a design change likely broke without pretending to generate CAD.

## Demo flow

1. Create a project
2. Upload before and after assemblies
3. Open the parsed part list
4. Run a comparison
5. Show changed parts
6. Show affected parts and severity
7. Open the visual overlay panel
8. Read the grounded summary
9. Export the report

## Demo line

We are not generating CAD. We are helping engineers understand what a change likely broke.
